# CANFD–USBC Adapter

**Components:**
- STM32G0B1CBT6
- TJA1051T-3
- AMS1117-3.3

**The goal is to develop the firmware myself using:**
- STM32CubeMX / IDE
- OpenOCD + ST-Link

The Board will also be tested using the candleLight Opensource Project:
- https://github.com/linux-automation/candleLightFD
- https://github.com/candle-usb/candleLight_fw

*The firmware development will be included in this Repo at a later point (once the PCB has been ordered).*
